execution_perf module

execution_perf.get_avg(times: list)[source]

Computes the average time for each size. The number of values per size is the same as ITERS :param times: list of all time measurements. The size of times needs to be len(args) * ITERS :return: list of time averages

execution_perf.get_max(times: list)[source]

Computes the maximum time for each size. The number of values per size is the same as ITERS :param times: list of all time measurements. The size of times needs to be len(args) * ITERS :return: list of the maximum time for each size

execution_perf.get_measurements(param: str, args: list)[source]

Computes and measures the execution time of the navier stokes spectral while varying param with the values in args :param param: the parameter to be varied in the navier stokes spectral function :param args: the values that param will use for its iterations :return: a list of all time measurements

execution_perf.get_min(times: list)[source]

Computes the minimum time for each size. The number of values per size is the same as ITERS :param times: list of all time measurements. The size of times needs to be len(args) * ITERS :return: list of the minimum time for each size

execution_perf.get_std(times: list)[source]

Computes the standard deviation for each size. The number of values per size is the same as ITERS :param times: list of all time measurements. The size of times needs to be len(args) * ITERS :return: list of standard devations

execution_perf.plot(x: list, y: list, std: list, param: str)[source]

Plots a graph of execution time with varying a variable in navier stokes spectral and with standard deviation applied. :param x: the size values used for the varying parameter :param y: the execution times of navier stokes spectral :param std: the standard devations of the execution times :param param: the varying parameter used :return: None
